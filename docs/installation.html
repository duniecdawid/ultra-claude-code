Installation

Get Ultra Claude running in your project in three steps.

1

Add the marketplace & install the plugin

In any Claude Code session, run these two commands:

# Add the marketplace
/plugin marketplace add duniecdawid/ultra-claude-code

# Install the plugin
/plugin install uc@ultra-claude

This registers the marketplace and installs Ultra Claude as a plugin. All skills become available as /uc:* slash commands.

Auto-updates: Enable automatic updates so the plugin stays current:

/plugin marketplace auto-update duniecdawid/ultra-claude-code
2

Initialize the documentation layer

Navigate to your target project and run the init skill to scaffold and configure everything:

/uc:init-project

This explores your codebase, scaffolds the documentation/ structure, derives .claude/ config files, and migrates any existing docs to canonical locations — all in one step. It enters plan mode so you can review everything before files are created.

Works for any project state. Greenfield, existing docs in non-standard layout, partially structured, or already set up. After init, run /uc:doc-code-verification-mode to verify and improve the generated docs.

3

Verify the installation

Run the help command to confirm everything is working:

/uc:help

This shows all available commands and guides you on which skills, agents, and workflows to use for your task.
